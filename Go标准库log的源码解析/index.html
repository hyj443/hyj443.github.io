type Logger

Logger 表示一个 active logging object
用来生成 lines of output to an io.Writer。
每个日志操作都会调用一次 io.Writer 接口的 Write 方法。
Logger 可以在多个 goroutine 中同时使用，它会保证对 writer 的顺序访问（sync.Mutex）

func Caller(skip int) (pc uintptr, file string, line int, ok bool)

runtime.Caller 返回 go 协程调用栈所执行的函数的文件和行号信息，skip 为上溯的栈帧数，0 表示 Caller 的调用者，返回相应调用的文件中的程序计数器、文件名、该调用在文件中的行号。如果无法获得信息，则 ok 为 false。

Write 将 len (p) 个 byte 的数据，从 p 写入底层的数据流，它会返回写入的字节数 n (0 <= n <= len (p))，以及遇到的导致写入提前结束的错误。如果返回的 n 小于 len (p)，则必须返回一个非 nil 的 error，Write 不能修改切片 p 中的数据，即使临时修改也不行。
